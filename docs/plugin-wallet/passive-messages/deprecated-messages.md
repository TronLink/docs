以下四个消息在未来4.x版本将会被废弃

  1. 用户拒绝连接消息`rejectWeb`

  2. 用户断连网站消息`disconnectWeb`

  3. 用户确定连接消息`acceptWeb`

  4. 用户主动连接网站消息`connectWeb`
